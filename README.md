# Smart Deal Finder - Frontend

A product discovery website for finding the best deals from Flipkart, Myntra & Amazon.

## Features

- Search products with filters (platform, category, brand, price, discount)
- Quick filter buttons
- Mobile responsive design
- Product cards with images, ratings, and affiliate links
